# 📊 HR Analytics – Workforce & Attrition Insights

## 🔍 Project Overview

This project analyzes employee data using **Power BI** to understand workforce composition, salary structure, performance, and key drivers of employee attrition.

The goal is to help organizations make **data-driven HR decisions** and reduce employee turnover.

---

## 🎯 Business Problem

Employee attrition leads to:

* Increased hiring and training costs
* Loss of productivity
* Reduced employee morale

👉 This project identifies **why employees leave** and provides **actionable insights to improve retention**

---

## 📂 Dataset & Tools

* 📁 Dataset: IBM HR Analytics Dataset
* 📊 Tool: Power BI
* ⚙️ Techniques:

  * Data Modeling (Star Schema)
  * DAX Measures
  * Data Visualization

## 🔥 Key Drivers of Attrition

* 📉 **Job Satisfaction**

  * Low (1) → **30.3% attrition**
  * High (4) → **13.9%**

* ⏱️ **Overtime**

  * Yes → **33.1%**
  * No → **10.8%**

* ⚖️ **Work-Life Balance**

  * Poor balance → highest attrition (~31%)

* 💼 **Job Role**

  * Sales Representative → **~39.8% attrition (highest)**

---

## 📌 Key Findings

* Overtime increases attrition **3x**
* Low satisfaction strongly impacts retention
* Entry-level salary group has highest attrition
* Specific roles are high-risk

---

## 💡 Business Recommendations

* Reduce excessive overtime workload
* Improve employee satisfaction programs
* Focus retention strategies on high-risk roles
* Improve compensation for lower salary bands

---

This project demonstrates how **HR analytics** can uncover hidden patterns in employee behavior and help organizations:
